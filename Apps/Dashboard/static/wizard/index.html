No apikeys.json

In order to run a locker yourself, you need to add in a file called apikeys.json that has the appKey and appSecret for at least one service. For example:

{"facebook": {"appKey": "YOUR-APP-KEY", "appSecret": "YOUR-APP-SECRET"}}

To get these keys, please see our wiki page: How to get API keys for use with a locker

Please add apikeys.json, and restart lockerd.js.
